The goal of this project is to give you a crash course in Java. Since CS61B is not intended to be a course about Java, we will be moving quickly, learning fundamental aspects of the Java language in just four weeks.
Ultimately, you will be creating a program NBody.java that draws an animation of bodies floating around in space tugging on each other with the power of gravity.

第一个project(叫第0个更恰当些)主要是快速适应java的编程环境，
因为cs61b是数据结构课程，本身也不会教什么看起来很炫酷的东西，虽然说我已经觉得够炫酷了。
这个proj做了大概有一天半，其实proj本身并不难，讲义给的可以说是十分详细了
不过这是我第一次做国外大学的proj，有很多值得注意的地方
首先给我感触最深的就是讲义的阅读量真的蛮大的，特别是对我这种英语渣，看懂倒是没什么问题，主要是阅读速度很捉急
然后就是给的skeleton真的很全面，几乎每一个方法都会给test，甚至让你自己写test
整个完成后还会给大量的数据和音频让你感受更大的快乐
感觉国外或许更重视多个学科之间的交叉吧，讲义一步步的带你推导物理公式，再转变成代码
终于找到了一点emmm计算机科学的感觉哈哈哈哈哈哈哈
其实，遇到最大的问题倒不是代码怎么写，而是不熟悉环境而导致的一个又一个bug
比如说调用StdDraw时出现“错误：编码GBK的不可映射字符”，找了两个小时才解决
还有调用show()方法时有出现过
“注: NBody.java使用或覆盖了已过时的 API。
注: 有关详细信息, 请使用 -Xlint:deprecation 重新编译。”
也是找了一段时间才解决

Anyway，这个proj让我真正热爱上了这门课，所以，加油吧！
